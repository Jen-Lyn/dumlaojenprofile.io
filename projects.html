Project - Library Information System

This was my project when I was in college it was my thesis. I and my groupmates created a Library Information system using Laravel Framework.

Home page

The user can select between home, search material, gallery and contact. The user can also login if the user has an account as admin.

Search material section

Users and guests can select type of sorting to display the result of the use's input.

Home reports section

User can viewed the report of most search material and most viewed material in the home page of the website.

View thesis and Download thesis

Viewing of a thesis' information with abstract. A student can also download digitized copy of thesis while quests are prohibited from downloading the pdf. The user can also view Information File, Serial with issues books.

Create and edit information file

The admin is presented with fields pertaining to the details of an information file. The admin may click the "save" button to save the inputted details. The admin can insert and edit info of thesis, information file and serial books with issues.
